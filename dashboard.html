Автоматизация

Автозаполнение спецификаций через Power Query + общий отчёт 1С

Устранил ручной перенос параметров товара из карточек номенклатуры в спецификации. Процесс, который требовал поиска по каждой позиции вручную и часто приводил к ожиданию ответа поставщика, теперь выполняется за секунды.

Было
Ручное заполнение
Десятки колонок (размер, вес, штрихкод, параметры) заполнялись вручную из карточек номенклатуры. В лучшем случае — 5 минут на позицию. В худшем — отправка пустой спецификации поставщику и ожидание 1–2 дня.
→
Решение
Power Query + выгрузка 1С
Реализовал выгрузку общего отчёта со всеми реквизитами товаров из 1С. При открытии файла Power Query подтягивает актуальные данные. Формула ВПР протянута на всех ячейках.
→
Стало
Артикул + цена — и готово
Для заполнения спецификации достаточно вставить артикул и цену. Поставщику остаётся только распечатать и подписать — 10 минут вместо цикла ожидания.
